A Group 协同编辑器
=========

独立于主项目，需单独安装,启动


```
bower install
npm install
noder server.js
```


url 参数说明

```
http://localhost:3000/editor?group=a&file=%E7%A1%AC%E7%9A%84%E6%B8%B8%E6%88%8F%E5%BC%80%E6%94%BE%E5%B9%B3%E5%8F%B0.md&debug#
```

小组名称 group
文件名称 file
是否开启调试 debug
